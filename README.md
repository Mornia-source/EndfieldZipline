# 终末地长距离滑索 · Endfield Zipline

适用于 **Terraria [tModLoader](https://github.com/tModLoader/tModLoader)** 的内容模组：在世界中放置滑索装置，在多个滑索点之间快速移动，玩法参考《明日方舟：终末地》中的长距离滑索体验。

**作者：** Mornia  
**模组显示名：** 终末地长距离滑索  
**内部名：** `EndfieldZipline`

## 功能概览

- 放置**长距离滑索**_tile，在已架设的滑索点之间往返移动  
- 多种**滑索架**材质与变体（如精金、钛金、机械、巨石等），适配不同进度与玩法  
- **全屏地图**上可选显示滑索位置图标  
- **客户端配置**：快速移动按键、索道绳索颜色、移动速度预设（含简易模式）、绳索物理视觉效果、音效开关等  

## 使用说明

1. **挂上滑索：** 右键滑索架顶端  
2. **快速移动：** 使用快捷键（默认 `E`，可在模组配置中修改）或右键另一个滑索点，向面朝方向移动至合适目标  
3. **脱离：** 按跳跃键或再次右键滑索架  

若两点之间有方块阻挡，移动会被阻止（游戏中会有相应提示）。

## 合成示例

在铁砧处可用以下材料一次合成 **2 个**长距离滑索（与 Steam 工坊说明一致）：

- 绳 × 50  
- 链条 × 10  
- 铁锭 × 5（配方组为原版铁锭类）  

## 配置说明

在 **模组配置 → Endfield Zipline Config** 中可调整例如：

- 快速移动按键（键名，如 `E`、`Q`、`F`）  
- 索道绳索颜色、是否重置为默认浅蓝色  
- 移动速度预设（默认 / 较快 / 极快 / 传送等）与简易模式  
- 全屏地图是否显示滑索图标、绳索摆动视觉效果、是否关闭滑索相关音效  

## 开发与构建

1. 安装与游戏版本匹配的 **tModLoader**  
2. 将本仓库置于 `Documents/My Games/Terraria/tModLoader/ModSources/` 下（或你的 ModSources 目录）  
3. 在 tModLoader 中 **开发 → 构建 + 重载** 即可  

依赖以项目中的 `EndfieldZipline.csproj` 及对 `tModLoader.targets` 的引用为准。
